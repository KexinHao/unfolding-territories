Checkmate
otherly > Anna Carpaij

Sound, vibration, light, touch, it asks for attention, it gets our attention.
Checkmate is a deconstruction of the experience of the phone. An object that we turned into something that is almost alive, it provides us with information, it makes it possible to be in contact with the world around as 24/7, it became a part of us and changed the value of time and space.
We live in symbioses with a non-living organism. An organism that we can fully control, we can give it new features, new shapes, a new color… by creating this organism, we put ourselves in the position of checkmate, we are not killed by this non-living organism but we cannot live without any more, so it wins.
